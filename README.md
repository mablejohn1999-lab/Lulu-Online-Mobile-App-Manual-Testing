
# Lulu Online Mobile App – Manual Testing Project
## Project Overview
This project focuses on **manual testing of the Lulu Online Mobile Application**, covering critical user workflows to ensure functionality, usability, and reliability.  
Testing activities included detailed test case design, execution across multiple modules, defect logging, defect analysis, and preparation of test reports.
---

## Application Under Test (AUT)
**Lulu Online Mobile App**  
Domain: E-commerce / Online Grocery & Retail  
Platform: Mobile Application (Android / iOS – Functional Testing Scope)
---

## Modules Covered
Manual test cases were designed and executed for the following **5 core modules**:

1. **Home**
2. **Categories&Happiness**
3. **Offers**
4. **Cart**
5. **Checkout**
---
## Test Artifacts Included
- **Test Cases** – 109 test cases (module-wise)
- **Bug Report**
- **Defect Distribution Report**
- **Test Report**

The following testing types were performed as part of this project:

- Functional Testing  
- UI Testing  
- Smoke Testing  
- Regression Testing  
- Negative Testing  
- Boundary Value Testing  
- Exploratory Testing
